# Akilesh Yelchuru Portfolio

A modern, responsive portfolio website built with HTML5, CSS3, and Vanilla JavaScript.

## 🌟 Features

- **Responsive Design** - Mobile-first approach that works on all devices
- **Smooth Scrolling** - Smooth navigation between sections
- **Modern UI** - Clean and professional design with gradient accents
- **Contact Form** - Functional contact form with validation
- **Mobile Menu** - Hamburger menu for mobile devices
- **Scroll Animations** - Engaging animations as you scroll
- **Fast Loading** - Lightweight with no external dependencies
- **SEO Friendly** - Semantic HTML structure

## 📁 Project Structure

```
├── index.html       # Main HTML file
├── styles.css       # CSS styling
├── script.js        # JavaScript functionality
└── README.md        # This file
```

## 🎨 Sections

1. **Navigation** - Sticky header with smooth navigation
2. **Hero** - Eye-catching landing section
3. **About** - Personal information and stats
4. **Projects** - Featured project showcase
5. **Skills** - Technical skills organized by category
6. **Contact** - Contact information and form
7. **Footer** - Social links and copyright

## 🚀 Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Customize the content to match your information

## 📝 Customization

### Update Personal Information
Edit the following in `index.html`:
- Name and title
- About section text
- Contact information (email, phone, location)
- Project descriptions and links
- Skills and technologies

### Modify Colors
Update the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #f5576c;
    /* ... more colors */
}
```

### Add Projects
Add new project cards in the projects section by copying and modifying the project card template.
